**查快捷键的快捷键：ctrl + alt + A**

![img](https://pic2.zhimg.com/80/v2-ea5a0b172bcffe58533d701211ba8c01_720w.jpg)

## **自动化类：**

万能辅助键：**alt + enter**

格式化代码：**Ctrl + Alt + L** ，可以对当前文件和整个包目录使用 

自动生成函数：**alt + insert** （和eclipse的alt + shift + s效果一样）

自动生成System.out.println()：**sout** 

自动生成变量名：**ctrl + alt + V** 

自动抛出异常：**ctrl + alt + T**

生成环绕代码：**shift + alt + z**，用来生成常见的类似try/catch之类的代码

自动补全：需要自己手动修改成**alt + /**，默认是ctrl + space

变量改名：**shift + F6**

方法参数提示：也需要自己设置成 **ctrl + P**

运行当前类：**ctrl + shift + F10**

抽取选中的代码行到一个方法中：**ctrl + alt + m**

调出粘贴板：**ctrl + shift + v**

## IDEA内置代码模板：

从0到100变量i的for循环：**100.fori**

以flag为条件判断：**flag.if**

输出成员变量flag：**flag.sout**

## 列出与显示类：

打开控制台：**alt + 4**

列出当前类方法集合：**Ctrl+F12** ，查看file，method结构图、类继承机构图

列出当前类依赖关系：**Ctrl+shift+Alt+U**   查看maven依赖，类图

列出方法调用层次：**ctrl + H**

## **查找类：**

查找类：**ctrl + n**，默认查找项目内的类，查找时再按一次则是查找包含项目外的类

查找文件：**ctrl + shift + n** ，默认查找项目内的文件，查找时再按一次则是查找包含项目外的文件

查找方法与属性：**ctrl + shift + alt + n，默认**查找项目内的方法&属性，查找时再按一次则是查找包含项目外的方法&属性

查找字符串：**ctrl + shift + f**

## **选择文本类：**

选择当前行：**ctrl + insert**

逐个向下选择相同字符：**alt + j**

在每行直接选择相同字符：**ctrl + shift + alt + j**

## 编辑文本类：

在光标下面插入新的一行：**shift + enter**

剪切光标所在的行：可以直接用**ctrl + x**

复制光标所在行，并黏贴到光标下面一行：**ctrl + d**

删除光标所在行：**ctrl + y**

移动代码行：**alt + shift + ↑ 或 ↓**

## Debug类：

添加断点：**ctrl+f8** 

添加条件断点：**ctrl+shift+f8** 

debug运行：**shift+f9** 

表达式求值：**alt+F8**
